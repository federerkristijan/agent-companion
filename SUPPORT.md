# Support

Thanks for using **Agent Companion**! Here's how to get help.

## Questions & help

- Start with the [README](README.md) and [PROJECT_BRIEF.md](PROJECT_BRIEF.md) — they cover the architecture, setup, and how the pieces fit together.
- For usage questions or discussion, open a [GitHub Discussion](../../discussions) (or a discussion-labelled issue if Discussions aren't enabled).

## Bugs & feature requests

- Open a [GitHub Issue](../../issues) with clear steps to reproduce, expected vs. actual behavior, and your environment.

## Security

- Do **not** report security vulnerabilities in public issues. See [SECURITY.md](SECURITY.md).
